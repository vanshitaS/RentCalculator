# RentCalculator
A Python-based tool developed in Jupyter Notebook to help users calculate and split rental costs, utilities, and deposits.

## Features
* Calculates monthly rent per person.
* Includes utility split logic (Electricity, Water, Internet).
* Visualizes spending trends (if applicable).

## How to Run
1. Clone this repository.
2. Ensure you have Jupyter installed.
3. Install dependencies: `pip install -r requirements.txt`
4. Run `Rent_Calculator.ipynb`.
